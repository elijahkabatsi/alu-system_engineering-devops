# SSH Project

This project covers the basics of SSH, including:

- What a server is
- Where servers are usually located
- What SSH is and how it works
- How to create an RSA key pair
- How to connect to a remote server using SSH and a private key
- How to configure the SSH client using ~/.ssh/config

## Files

- 0-use_a_private_key — connects to a server using a specific private key
- 1-create_ssh_key_pair — generates an RSA SSH key pair
- 2-ssh_config — configures the SSH client to use a specific identity file
