LU System Engineering & DevOps Project

This repository contains scripts and configurations for various system engineering and DevOps tasks as part of the ALU curriculum. The project is organized into three main folders:

## Folder Structure

### 1. `load_balancer`
Contains scripts and configuration files related to setting up and managing load balancers. These scripts help distribute traffic across multiple servers to ensure high availability and reliability of web applications.

### 2. `ssh`
Contains scripts for SSH key management, server access automation, and secure remote connections. These scripts ensure proper authentication and secure management of server connections.

### 3. `web_server`
Contains scripts and setup instructions for web server deployment. This includes setting up domains, configuring web services, and ensuring web servers are running properly on Ubuntu systems.

## Requirements
- Ubuntu 16.04 LTS / 20.04 LTS
- Bash scripting knowledge
- Executable permissions on all scripts
- Shellcheck (version 0.3.7) for script validation

## Notes
- All scripts should start with `#!/bin/bash` and end with a newline.
- Follow folder-specific instructions for executing scripts.
- This repository is part of the ALU System Engineering & DevOps projects and adheres to all course submission guidelines.
